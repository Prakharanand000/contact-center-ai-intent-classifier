MCP Server Integration

Expose this classifier as a Model Context Protocol (MCP) tool - connect it directly to Claude Desktop, Cursor, or any MCP-compatible agent.

What is MCP

Model Context Protocol

MCP lets AI assistants like Claude call external tools as part of their reasoning. By exposing this classifier as an MCP server, Claude can automatically classify customer intents mid-conversation, no copy-pasting required. Same architecture as WatchMyMoney's native MCP server.

Claude DesktopCursorLangChainAny MCP Client
Setup Steps
1

Start the API locally

The API must be live on port 8000 before Claude can call it.

python -m uvicorn api.main:app --reload --port 8000
2

Add to Claude Desktop config

Open %APPDATA%\Claude\claude_desktop_config.json and add:

{ "mcpServers": { "contact-center-ai": { "command": "python", "args": ["-m", "mcp_server"], "cwd": "C:/Users/prakh/Downloads/Code & Projects/contact-center-ai-intent-classifier" } } }
3

Call from any Python agent

import requests response = requests.post( "http://localhost:8000/predict", json={ "text": "I need to cancel my subscription", "session_id": "session_abc123" } ) result = response.json() print(result["prediction"]["intent"]) # cancel_subscription print(result["prediction"]["confidence"]) # 0.9821
4

Tool schema for MCP clients

{ "name": "classify_intent", "description": "Classify a customer utterance into one of 151 intents", "inputSchema": { "properties": { "text": { "type": "string" }, "session_id": { "type": "string" } }, "required": ["text"] } }
